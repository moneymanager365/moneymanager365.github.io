Life is much easier when you have good financial skills. How you spend your money impacts your credit score and the amount of debt you end up carrying. If you’re struggling with money management issues such a living paycheck to paycheck despite making more than enough money, then here are some tips to improve your financial habits.

 

How To Manage Your Money Better with Money Manager 365
1) Account

Account is used to differenship your bank accounts. For example, you can create Bank A, Bank B, and Bank C. And use these three accounts to manage your expenses. By default, the system will create a "Cash" account for you.

 - If you delete the account, all the transactions or recurring transactions belong to this account will be deleted too.

 - Changing the account currency would not affect any transaction, but it may change the decimal places display on the transaction amount. 

 - Changing the "Opening Balance" will cause the system recalculate the account balance. If you think the balance is not correct, then you can try to change the "Opening Balance", then the system will trigger the balance re-calculation.

 - "Auto Select", by default the system will set the Auto Select function for your first account (Cash), if you have multiple account wish to select the account yourself, you can uncheck the "Auto Select" from account setting.

 - "Share", you can share the particular account to your family, and they will be able to add transaction to this shared account, and all the members can see the total amount spend on this account. This is when you want to organise a trip and everybody can see how much they spend on this trip. For the security, no body can change other members transactions.

 

2) Category

Category is used to organise your expenses in daily basic. You can create new category base on your need. You also can create multiple sub categories to help you in money management.

- If you delete the category, it would not affect anything, and would not delete any transaction.
- If you change the category name, then the system will scan all the transactions with the same category name and change the category name accrodingly.
- If you change the category color, then the system will scan all the transactions with the same color and change the color accrodingly.

 

3) Budget

One of the powerful tool that help you manage your money. You can create a new budget base on different accounts, category, date range, and target amount. 

You can create a budget for your expense, or even for your income. The system will automatically sum up all the amount base on the category.

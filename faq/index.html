Money Manager 365 - FAQ

1. What is the support email?

You can send an email to our support team - moneymanager365@gmail.com

 

2. What happens if I changed the account "Opening balance"?

If you changed the opening balance of an account, the system will re-calculate the balance on each day.

 

3. What happens if I changed the currency code?

If you changed the currency code, it wouldn't affect any transaction amount, the system would not update the amount.

 

4. What happens if I deleted the category?

if you deleted a category, it wouldn't affect any transactions or accounts.

 

5. What happens if I deleted the account?

All the transactions and repeat transactions below to this account will be deleted.

 

6. I've multiple devices, how do I sync the data across all my devices?

You can create a cloud account and then log in to the cloud account on each device, the system will automatically sync all the data.

 

7. How do I import the data?

You can import the data from the CSV file.

 

9. Can I share an account with my family members, but they are using a different cloud account?

Yes, you can share your account with other people, regardless of the different cloud account, and you can share with as many as you want.

 

10. Can I use the app without a cloud account?

Yes, you can use the app without a cloud account, and even without internet access, the app can run offline.

 

11. I don't want to use multiple accounts, I just want to log my expenses only.

Yes, you can go to account setting, and then choose "Auto select", and every time you log the expenses, the system will automatically select the account for you.

 

12. What happens if I deleted the budget?

if you deleted a budget, it wouldn't affect any transactions or accounts.
